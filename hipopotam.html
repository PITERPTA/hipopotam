Hipocykloida

Hipocykloida to krzywa płaska, którą tworzy punkt na obwodzie mniejszego okręgu toczącego się bez poślizgu wewnątrz większego okręgu.

\begin{cases}x(t) = (R-r)\cos t + r\cos(\frac{R-r}{r}t) \\y(t)=(R-r)\sin t - r\sin(\frac{R-r}{r}t)\end{cases}

gdzie \(t\) jest parametrem opisującym ruch punktu po krzywej

Okrąg

Krzywe Lissajous w p5.js

\begin{cases} x(t) = \cos(at) \\ y(t) = \sin(bt) \end{cases}

Python


      import numpy as np
import matplotlib.pyplot as plt

# Parametry hipocyklody
R = 1  # promień koła
r = 0.3  # promień okręgu, który się toczy
theta = np.linspace(0, 4 * np.pi, 1000)  # kąt (od 0 do 4*pi)

# Wzór na hipocykloidę
x = (R - r) * np.cos(theta) + r * np.cos((R - r) / r * theta)
y = (R - r) * np.sin(theta) - r * np.sin((R - r) / r * theta)

# Rysowanie wykresu
plt.figure(figsize=(6, 6))
plt.plot(x, y)
plt.title("Hipocyklodia")
plt.axis("equal")
plt.grid(True)
plt.show()
